The goal of this project is to build a **classification model** that can assist doctors and patients by providing quick, data-driven predictions of breast cancer using the **Breast Cancer Wisconsin dataset**.

The web interface allows users to:

- Get an instant prediction result (Malignant or Benign)
- View predictions with a clean and user-friendly UI
